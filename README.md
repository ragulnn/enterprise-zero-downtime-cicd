# Enterprise Zero-Downtime CI/CD Platform

## Overview

This project demonstrates a production-grade CI/CD platform on Azure Kubernetes Service (AKS).

## Features

- Azure DevOps CI/CD
- Azure Kubernetes Service
- Terraform Infrastructure
- Azure Container Registry
- Helm
- Argo Rollouts
- Canary Deployment
- Blue-Green Deployment
- Zero Downtime Deployment
- Prometheus
- Grafana
- Alertmanager
- DevSecOps
- Automated Rollback
- Performance Testing using K6
